# BUG-003. `/sdx:switch` делает `git add -A` с авто-коммитом

## Суть
Слепой `git add -A && git commit` перед checkout закоммитит в ветку сессии всё,
что лежало в дереве (мусор, потенциально секреты), с шумным сообщением. Не обработаны
конфликтные случаи (checkout невозможен, та же ветка).

## Рекомендация
Минимум — `git stash push` вместо авто-коммита; лучше — перевести параллельные сессии на
`git worktree` (одна сессия = свой worktree): убирает класс проблемы «переключение через
общее дерево». Хуки уже worktree-совместимы (сессия по ветке).
