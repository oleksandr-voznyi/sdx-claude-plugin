# DEBT-012. `@protocol.md` инжектится каждой командой

## Суть
Все 13 команд включают полный `protocol.md` (~170 строк) при каждом вызове;
за сессию с 5–6 командами — заметный повтор.

## Рекомендация
Тонкие команды (`status`, `checkpoint`, `switch`, `backtrack`) ссылаются
на протокол текстом («прочитай при необходимости»); `@`-инжект оставить у команд, которым
нужен целиком (`start`, `next`, `archive`, `verify`).

## Резолюция
Закрыто (сессия `fw-econ-a4d1-20260703`, 2026-07-03) — тонкие команды (`status`,
`checkpoint`, `switch`, `backtrack`) ссылаются на протокол текстом без `@`-инжекта; `@`
сохранён у команд, которым нужен весь протокол.
