# DEBT-020. Каталоги разборов без индекса; формулировка ADR-017 разошлась с фактом

## Суть
ADR-017 явно фиксирует, что собственного индекса (в отличие от `docs/backlog/README.md`)
для `docs/history/{retro,postmortem,intake}/` не заводится. Фактически в каждом каталоге
лежит `README.md` — по назначению это файл-заглушка для версионирования пустого каталога,
а не индекс, но буква ADR и содержимое репозитория внешне расходятся.

Содержательная же часть в силе: при накоплении разборов искать нужный будет неудобно —
ни списка, ни группировки по периоду/теме нет.

## Рекомендация
Либо уточнить формулировку ADR-017 («заглушка — не индекс»), либо превратить заглушки в
настоящие индексы с автопополнением на Closeout (пункт 4 чек-листа уже переносит документ
разбора — дописать строку в индекс тем же шагом).
