# FEAT-001. Англоязычная документация для GitHub-аудитории

## Суть
Репозиторий плагина публичен на GitHub, но вся пользовательская документация — на русском
(по языковой политике CLAUDE.md §1); англоязычная аудитория не может ознакомиться с
плагином: нет ни английского README, ни описания репо (About), ни английских заметок
в релизах.

## Рекомендация
Полный английский перевод README отдельным файлом `README.en.md` с взаимными ссылками
(русский остаётся каноническим); англоязычное About репозитория; английский вариант
в описаниях GitHub-релизов (двуязычные заметки). Синхронную поддержку двух README
закрепить процессно. Дальнейшее развитие мультиязычности — FEAT-002.

## Резолюция
Закрыто сессией `fw-readme-en-20260720` (2026-07-20): создан `README.en.md` (полный
перевод, взаимные ссылки, нота о рабочем языке), About репозитория переведён на
английский, заметки релизов v1.0.0–v1.2.0 сделаны двуязычными; попутно актуализирован
README (14 команд, `docs/backlog/` в per-project слое). Продолжение — FEAT-002.
