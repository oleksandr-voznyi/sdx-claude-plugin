# IDEA-001. REQ-CACHE-1 (Фаза 2) остаётся актуальным

## Суть
Требование REQ-CACHE-1 (Фаза 2 roadmap) — детерминированный порядок сборки контекста —
не реализовано и остаётся актуальным; без него появление fanout-контура ломает
кэшируемость контекста. Исходный roadmap живёт в gitignored-бандле (см. DEBT-008).

## Рекомендация
Зафиксировать детерминированный порядок сборки контекста ДО появления fanout-контура
(IDEA-002), как планировалось в Фазе 2. Полная формулировка и дизайн-срез —
`docs/specs/phases-2-4-deferred.md` (раздел REQ-CACHE-1).

## Резолюция
Отложено (roadmap, Фаза 2) — актуально до появления fanout-контура (IDEA-002).
Полная формулировка промоучена в `docs/specs/phases-2-4-deferred.md` (сессия
`fw-roadmap-20260720`, закрытие DEBT-008).
