# PROC-006. Публичность и трекшн: путь к программе Claude for Open Source

## Суть
Программа Anthropic «Claude for Open Source» (6 месяцев Claude Max 20x) требует измеримой
востребованности проекта, которой у SDX пока нет: планки — 5000+ звёзд репо, либо 500+
зависимых репозиториев / 200k+ скачиваний в месяц (для реестров), либо 20+ внешних
контрибьюторов за год; есть оговорка «critical infrastructure» («поддерживаете то, от чего
экосистема тихо зависит — подавайтесь»), но и она подразумевает реальное стороннее
использование. Репозиторий публичен с 2026-07-19, контрибьютор один, внешних
пользователей нет. Критерии программы — готовый roadmap публичности проекта.

## Рекомендация
Наращивать публичность и трекшн по нарастающей, переиспользуя критерии как чек-лист:
- [x] публичный репозиторий, MIT, релизы с тегами и заметками (2026-07-20);
- [x] английский README + About + двуязычные релизы (FEAT-001);
- [ ] `CONTRIBUTING.md` + Code of Conduct;
- [ ] CI на GitHub Actions (тесты хуков уже есть — повесить прогон + бейдж);
- [ ] примеры/скринкаст полного рабочего цикла SDX (start → verify → archive);
- [ ] подача в community-каталоги плагинов Claude Code и awesome-списки;
- [ ] снятие языкового барьера для внешних пользователей (FEAT-002);
- [ ] первые внешние пользователи/issue/PR — метрики трекшна;
- [ ] через 6–12 месяцев при реальной трактции — заявка через оговорку
      «critical infrastructure» (https://claude.com/contact-sales/claude-for-oss).
Параллельный личный путь (не зависит от SDX): критерий «active contributor» —
100+ смёрженных PR в чужие репозитории за 12 месяцев.
